FlashShare is a high-performance, cross-platform file transfer and collaboration tool built with Flutter. It utilizes a local HTTP server and WebSockets to enable blazing fast, offline P2P communication between devices (Mobile-to-Mobile, Mobile-to-PC) without requiring an internet connection.
Beyond simple file transfer, FlashShare features a suite of productivity tools including FlashCast (Real-time Presentation), Phone Clone, and Universal Clipboard.
